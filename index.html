Setting Hugo Website & Host on Github Quick Usage Instructions Edit/add new content, this will automatically create new article in “content” directory. hugo new <subdirectory_name>/<articlename.extension> Ex. hugo new posts/new_post.md Push everything from project root “hugoWebsite” to Github git status git add . git commit -m "added new article" git push origin main Render the content to “public” directory hugo -D Go to “public” directory and commit everything to GitHub cd public
